# SitePortNeuf

Page d'accueil du **Domaine du Grand-Portneuf** — React + TypeScript + Vite.

## Démarrage

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:5173](http://localhost:5173).

## Build production

```bash
npm run build
npm run preview
```

## Structure

- `src/components/` — sections de la page (Header, Hero, Villas, etc.)
- `public/images/` — images du design
- `src/index.css` — styles du design Manus
